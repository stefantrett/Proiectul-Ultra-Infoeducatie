Proiectul Ultra

Intel Galileo


Intel Galileo este o placă de dezvoltare Open Source, bazată pe procesorul System-on-Chip, Intel Quark X1000, de cinci ori mai mic decât procesorul Intel Atom şi cu consum de 10 ori mai redus decât acesta. Galileo este perfect compatibilă hardware şi software cu platforma Arduino, un standard pentru entuziaşti, artişti, designeri şi pasionaţi de ştiinţă, interesaţi să dezvolte proiecte de tip „do-it-yourself“. Aceste proiecte necesită resurse minime din partea dezvoltatorilor şi au aplicabilitate în domenii precum automatizări, robotică, meteorologie, sănătate sau controlul gesturilor. Pentru dezvoltarea de proiecte cu Intel Galileo se poate folosi Linux, Mac OS şi Microsoft Windows.

Lucrarea a fost realizată pe un computer, cu procesor Intel inside CORE™ i5 cu 2,4 GHz și memorie RAM 6GB și sistemul de operare Windows 8.1. Codul sursă a fost creat pe platforma de dezvoltare Intel Galileo Gen 1 cu procesorul Intel Quark SoC X1000 cu o arhitectura x86, în mediul de dezvoltare Eclipse. Intel Galileo dispune de sistemul de operare Linux Yocto fără interfață vizuală, si este un mediu de dezvoltare semi-dependent de PC, prin intermediul calculatorului accesăm prin SSH linia de comanda a microcontrolerului.

Scopul

Scopul lucrării îl constituie construirea, programarea și prezentarea unui sistem inteligent de detecție a emoțiilor.

Proiectul Ultra este programat astfel încât să detecteze una dintre emoțiile aflate în baza de date și să realizeze un anumit lucru în funcție de emoția detectată. Acest sistem hardware și software ar putea fi implementat atât într-un sistem de securitate al casei care prima data iți recunoaște fața, după care transmite casei emoția ta, sau mai poate fi implementat în diferite proiecte: ceasul deșteptător care se oprește din sunat doar când zâmbiți, mașina inteligentă care în funcție de emoție pune o anumită melodie, iar dacă observă că șoferul este adormit îl atenționează cu un semnal sonor, etc.

Proiectul Ultra este un sistem inteligent de detecție a emoțiilor și a fețelor, programat în 5 limbaje: C++, Python, JavaScript, Html, CSS. Limbajul de programare C++ este folosit împreuna cu librăria OpenCV și se ocupa cu detecția feței, decuparea acesteia și determinarea emoției in funcție de baza de date; Primul cod scris in Python este folosit pentru decuparea feței, în funcție de coordonatele ochilor, care urmează a fi pusă în baza de date, iar al doilea cod scris în Python este folosit pentru crearea unui fișier CSV care are drept scop imitarea unei baze de date. Proiectul Ultra dispune și de o pagină web care rulează local, în rețeaua la care este conectat microcontrolerul și are rol de site de prezentare al sistemului. Pentru site am folosit 3 limbaje de programare enumerate mai sus. În primul limbaj de programare (HTML) am creat scheletul siteului, în JavaScript am programat apăsarea butoanelor și feedbackul oferit de acestea, iar în CSS am programat designul.
